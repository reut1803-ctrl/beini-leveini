# ביני לביני 🌸

מחברת דיגיטלית אישית - מקום לשפוך את הלב, בדיוק כמו המחברת המודפסת.

## איך מפעילים את זה (בלי שום התקנות, ישר מהטלפון)

1. נכנסים לעמוד הזה למעלה, לוחצים על **Settings** (הגדרות המאגר).
2. בתפריט בצד שמאל לוחצים על **Pages**.
3. תחת "Build and deployment" -> "Branch", בוחרים **main** ותיקייה **/ (root)**, ולוחצים **Save**.
4. מחכים כדקה, ואז מרעננים את הדף. יופיע למעלה קישור כחול בסגנון:
   `https://reut1803-ctrl.github.io/beini-leveini/`
5. פותחים את הקישור הזה בטלפון (אפשר לשמור אותו למסך הבית, כדי שיהיה כמו אפליקציה).

זהו, המחברת שלכם מוכנה ופעילה!

## איך זה עובד

- לוחצים על השער כדי לפתוח את המחברת.
- לוחצים על כפתור ה-**✚** הזהוב כדי להוסיף דף חדש.
- כל דף מקבל שורת השראה קטנה למעלה, ומתחתיה שורות של מחברת אמיתית לכתיבה חופשית.
- אפשר לדפדף בין הדפים עם החיצים למטה, או להחליק עם האצבע על המסך.
- כדי למחוק דף - לוחצים על ה-✕ הקטן שמופיע בפינה של הדף.
- הכתיבה נשמרת אוטומטית בטלפון/בדפדפן שבו נכנסתם (localStorage).

## גיבוי - חשוב לדעת

הכתיבה נשמרת מקומית בדפדפן שממנו נכנסתם. אם תנקו את היסטוריית הדפדפן, תעברו טלפון, או תיכנסו מדפדפן אחר - הדפים לא יעברו איתכם אוטומטית.
לכן, בעמוד השער יש קישור "שמירת גיבוי של הכתיבה שלי" - לוחצים עליו מדי פעם, וזה מוריד קובץ טקסט עם כל מה שכתבתם, לשמירה בטוחה.

## מבנה הקבצים (לסקרנים)

- `index.html` - השלד של האפליקציה
- `style.css` - כל העיצוב (הפרחים, הצבעים, השורות)
- `app.js` - הלוגיקה: הוספת דפים, מחיקה, שמירה
- `assets/` - הציורים הקטנים של הפרחים ברקע
